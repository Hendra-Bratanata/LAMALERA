# Tabel Broker Saham Indonesia: RITEL vs SHARK

## Ringkasan

Dokumen ini berisi daftar lengkap broker saham di Indonesia yang dikategorikan menjadi:
- **SHARK (Institutional/Smart Money)** - Broker untuk investor institusional
- **RITEL (Retail Investors)** - Broker untuk investor individu

## 3. Perbedaan Karakteristik SHARK vs RITEL

| Karakteristik | SHARK (Institutional) | RITEL (Retail) |
|:---|:---|:---|
| **Nilai Transaksi** | Ratusan TRILIUN | Milyaran |
| **Rata-rata/Transaksi** | Milyaran Rupiah | Ribuan/Jutaan Rupiah |
| **Frekuensi Harian** | Rendah (100-1000x) | Tinggi (1000-10000x) |
| **Jenis Nasabah** | Institusi, Asing, Fund Manager | Individu, Ritel |
| **Gaya Trading** | Swing, Long-term, Fundamental | Intraday, Scalping |
| **Platform** | Professional/Desktop | Mobile/App |
| **Fee** | Dapat dinegosiasi | Fixed standar |
| **Market Impact** | Tinggi (bisa gerakkan harga) | Rendah |
| **Analisis** | Fundamental mendalam | Teknikal/News driven |
| **Minimal Deposit** | Besar (ratusan juta+) | Kecil (Rp0 - Rp100rb) |

## 5. Rekomendasi Berdasarkan Jenis Investor

| Jenis Investor | Broker Paling Cocok | Alasan |
|:---------------|:--------------------|:-------|
| **Pemula banget** | Ajaib (XC), IPOT (PD) | UI mudah, gratis maintenance |
| **Social Trading** | Stockbit (XL), Sinarmas (DH) | Komunitas aktif |
| **Day Trader** | Mirae (YP), Maybank (ZP) | Eksekusi cepat |
| **Investor Jangka Panjang** | IPOT (PD), Stockbit (XL) | Robo trading, lengkap |
| **Nasabah Bank Mandiri** | Mandiri Sekuritas (CC) | Terintegrasi |
| **Nasabah Bank BCA** | BCA Sekuritas (SQ) | Terintegrasi |
| **Nasabah Bank BNI** | BNI Sekuritas (NI) | Terintegrasi |
| **Mau ikut Smart Money** | Pantau AK, CC, BK, GW | Tracking shark |

---

## 6. Tips Menggunakan Tabel Ini

1. **Mau ikut Smart Money?** → Pantau broker AK, CC, BK, GW
2. **Mau tahu sentimen ritel?** → Lihat XL, YP, PD, XC
3. **Trading harian?** → Pilih broker ritel dengan eksekusi cepat
4. **Investasi jangka panjang?** → Bisa ikut pola shark
5. **Cari broker di aplikasi** → Gunakan kode broker untuk filter
